# IOT-SmartCity

## 编译环境

- Java jdk8
- Android Studio 4.1.3
- 数据库是Android自带的Sqlite数据库

## 功能需求：

- 登录/注册
- 环境气象
- 智能商超
- 预警信息
- 智能园艺
